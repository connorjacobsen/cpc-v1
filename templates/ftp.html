FTP Instructions

Below are three ways to access the FTP server that has been created for our customers so that they can upload large files. This FTP server allows us to receive your files that are too large to be sent via email and that would otherwise have to be hand delivered on a USB drive, CD, or other media. Please view the methods below and choose the one that best fits your needs.

To obtain the username and password information, please contact us at (850) 893-4799 or via email at sales@cpccpc.com
  1. You can connect to the FTP server using any standard FTP client such as FileZilla, FTPVoyager, CoreFTP, or CuteFTP. To connect using an FTP client, you will need the following information:

    • Host/Server: files164.cyberlynk.net
    • Username: Please contact us for the username
    • Password: Please contact us for the password
    • Connection Type: FTP
    • Port Number: 21

    Once you have connected, you can upload files in a variety of ways, depending on the FTP client used. You can drag/drop, double click, or use the FTP client's built in icons or other methods to transfer over your file.

    If you do not have an FTP client installed, you can obtain the free, open-source FileZilla FTP Client by going to http://filezilla-project.org/.


  2. You can connect using Internet Explorer, or any other Web Browser by using the information below:
    1. First contact us to obtain the username and password information needed to connect.
    2. Then open your web browser, and in the URL box, enter in the following information:

      3. ftp://username:password@files164.cyberlynk.net

      Note that the words username and password in that link above will need to be replaced with the username and password you obtain from us.

    3. Once you have successfully connected, you should see a list of files. This should allow for dragging/dropping capabilities on any Windows machine.

  3. You can also use the Web Based JAVA FTP client on any Windows computer:

    4. http://files164.cyberlynk.net/client

    Again, please contact us to obtain the username and password information.

If you have any questions or problems, please feel free to contact us at (850) 893-4799 or via email at sales@cpccpc.com.
